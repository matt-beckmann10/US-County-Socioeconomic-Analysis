D502 Capstone Project
Analysis of Socioeconomic Factors Associated with County-Level Crime Rates in the United States


Author:
Matthew Beckmann

Project Description
-------------------
This project examines the relationships between selected socioeconomic variables and county-level violent crime and property crime rates across the United States using publicly available government datasets.


Project Contents
----------------
- Task3_Paper.docx
Final written capstone report.

- Crime_Analysis_Notebook.ipynb
Jupyter Notebook containing exploratory data analysis, statistical visualizations, Pearson correlation analyses,       multiple linear regression models, and written interpretations.

- python_scripts/
Python scripts used to acquire, clean, validate, merge, and prepare the datasets for analysis.
Scripts are numbered in the order they were executed.

- data_raw/
Original datasets obtained from:
* FBI Crime Data Explorer
* U.S. Census Bureau American Community Survey (ACS)
* Bureau of Labor Statistics (BLS)

- data_clean/
Cleaned datasets produced during preprocessing, including the final merged analysis dataset used in the notebook.


Project Workflow
----------------
1. Acquire raw datasets.
2. Clean and validate each dataset.
3. Merge datasets using county FIPS codes.
4. Produce the final analysis-ready dataset.
5. Perform exploratory data analysis.
6. Conduct Pearson correlation analysis.
7. Build multiple linear regression models.
8. Interpret results and summarize findings.

Notes
-----
The Python scripts and Jupyter Notebook were developed using Python. The notebook is provided with all outputs displayed so the complete analysis can be reviewed without re-running the code.
